UNIT 01
統計學導論

統計學是什麼？

統計學是關於對社會經濟現象數據資料的獲取、整理、分析、描述和推斷方法的總稱。包括獲取數據資料的方法，以及妥善利用這些數據的方法。

統計學在研究中扮演的核心角色：幫助研究者從有限的資料中，做出有根據的判斷與推論。日常生活中隨處可見統計的應用——從選舉民調、醫療臨床試驗，到行銷調查，皆仰賴統計方法。

敘述統計 vs 推論統計

敘述統計（Descriptive Statistics）

如果我們的興趣只限於手頭現有的數據，而不準備把結果用來推論群體，則稱為敘述統計。

例：消費者問卷的調查結果本身
推論統計（Inferential Statistics）

任何對數據（即樣本）的處理導致預測或推論群體的統計，稱為推論統計。

例：利用消費者問卷的調查結果，預測未來會購買的商品
💡 著名案例——尿布與啤酒：超市透過分析龐大的銷售資料，發現「購買嬰兒尿布的消費者，常常同時購買啤酒」，從而調整貨架陳列，提升銷售額。這是推論統計應用在商業決策的經典實例。

統計與科學方法

科學研究遵循以下循環步驟：

  • 觀察（Observation）：科學家蒐集、研究與其問題有關的事實，因而必須觀察研究對象發生的情形。
  • 假說（Hypothesis）：科學家解釋所觀察的事實，然後將其預感或直覺轉化為假說或理論，以表達洞察資料的結果。
  • 預測（Prediction）：科學家從假說及其相關理論中推演出某些結果。若假說成立，則此結果是從假說得出的新知識。
  • 驗證（Verification）：科學家蒐集新事實（即觀察）去檢驗由假說或理論所推測的結果是否成立。這個步驟使上述三個步驟必須重複進行。

UNIT 02
母體與樣本

核心概念定義

母體（Population）：它是一群具有某種共同特性的人或事物之個體所構成的群體。每一群體通常含有多種特性，可依某種特性歸納為一母體，亦可依另一種特性歸納為另一母體。
樣本（Sample）：從母體中依照一定方法抽取的部分個體，用來代表並推論母體特性。樣本的品質取決於抽樣方法的公正性與代表性。

母體參數（Parameter）
用來代表母體特徵的量值。

符號：μ（mu）

例：112年度A醫院所有新生兒之平均體重
樣本統計量（Statistic）
用來代表樣本特徵的量值。

符號：X̄（X-bar）

例：從A醫院新生兒中隨機抽出200位之平均體重
🔑 核心邏輯：我們通常無法直接量測母體（成本太高或不可能），因此透過抽取樣本、計算統計量（X̄），來估計母體參數（μ）的真實值。這正是統計推論的精髓。

UNIT 03
統計變數與測量尺度

變數與變量

統計資料一般是由一個或多個變數的值所組成：

  • 變數（Variable）：一種具有不同值或結果的特徵之衡量，凡一切可計量的特徵都稱為變數。例：身高、性別、教育程度、收入。
  • 變量（Value）：變數的值或量稱為變量。例：身高的變量可以是160cm、175cm等具體數值。

自變數 vs 因變數

自變數（Independent Variable）
被認為是「因」（cause），或用來解釋其他變數的變數。

在 y = f(x) 中，x 為自變數。
因變數（Dependent Variable）
被認為是「果」（effect），或是被解釋的變數。

在 y = f(x) 中，y 為因變數。
收入（因變數） = 教育程度 + 職業類型 + 性別 + 居住地區 + ε

四種測量尺度（Scale）

尺度決定了我們可以對資料進行哪些數學運算，也決定了應使用哪些統計方法。

尺度 類型 可用運算 特性 範例
名目尺度
Nominal		 非計量 = ≠ 僅能區分類別，無大小順序 性別、職業、語言、居住地區
順序尺度
Ordinal		 非計量 = ≠ > < 有大小順序，但間距不相等 教育程度（小學＜國中＜高中＜大學）
區間尺度
Interval		 計量 = ≠ > < + − 間距相等，但無絕對零點 顧客滿意度、溫度（℃）
（行銷學中假設滿意度各尺度間皆為等距）
比例尺度
Ratio		 計量 = ≠ > < + − × ÷ 有絕對零點，可進行所有四則運算 價格、年齡、所得、體重

UNIT 04
誤差、顯著性與信效度

常態分配與68-95-99.7法則

許多自然與社會現象的資料分布呈現鐘型曲線（Bell-shaped Curve），也稱常態分配（Normal Distribution）。此分配具有以下特性：

  • 以平均數為中心，左右對稱
  • 平均數 = 中位數 = 眾數
  • 曲線下方面積代表機率（總面積 = 1）
圖 4-1　常態分配的68-95-99.7法則

p 值與統計顯著性

大家習慣採用 0.05 當作一個臨界，當研究的 p 值小於這個臨界值的時候就宣稱研究結果達到統計顯著，也就是說大家普遍同意接受 5% 犯錯的可能性。
100% − 5% 犯錯 = 95% 信心水準

這個願意接受的犯錯可能性稱為「顯著水準（Significant Level）」——超過這個水準就宣稱研究達到統計顯著。

符號意義信心水準
*p < 0.05達到基本顯著水準95%
**p < 0.01高度顯著99%
***p < 0.001極高度顯著99.9%
🧠 批判性思考
  • p 值越小，表示該研究的結果越好嗎？
  • 樣本數越大，研究的 p 值一定越小嗎？

型一錯誤 vs 型二錯誤

假設飛機的鉚釘「足夠強壯」為虛無假設（H₀），以下說明四種結果：

真實情況：鉚釘足夠強
真實情況：鉚釘不夠強
測試結果
✓ True Positive（真陽性）
測試結果：足夠強。這是我們希望看到的結果，無法拒絕虛無假設（正確）。
✗ False Positive（偽陽性）
型一錯誤 Type I Error (α)
測試結果：足夠強，但實際上不夠強。錯誤地無法拒絕虛無假設，認為鉚釘足夠強，但實際上不夠強壯。
✗ False Negative（偽陰性）
型二錯誤 Type II Error (β)
測試結果：不夠強，但實際上足夠強。錯誤地拒絕了虛無假設，認為鉚釘不夠強，但實際上是夠強壯的。
✓ True Negative（真陰性）
測試結果：不夠強。這是我們希望看到的結果，正確地拒絕了虛無假設。

信度與效度

信度（Reliability）
測量的穩定性與一致性。若重複測量同一對象，每次結果都相近，則信度高。

類比：每次射擊都落在同一區域（不論是否命中靶心）。
效度（Validity）
測量工具是否真正測量到它想測量的概念（準確性）。

類比：射擊是否命中靶心，是否測量到真正想測量的東西。

Q4.請說明：為什麼「樣本數越大，p 值不一定越小」？
參考答案：p 值反映的是「在虛無假設為真的情況下，觀察到這樣極端結果的機率」。樣本數增大確實會提高統計檢定力，使真實存在的效果更容易被偵測到（p 值降低）；但若兩組之間本來就沒有差異，無論樣本數多大，p 值都不會顯著降低。換言之，p 值取決於效果的大小與樣本數的共同作用，而非單純由樣本數決定。
UNIT 05
彙總資料：表格與圖表

民意曲線的常見形態

在社會科學研究中，資料分布常呈現以下幾種典型形態：

  • 鐘型曲線（Bell-shaped Curve）：正態分配，資料集中於中間，兩端逐漸減少。符合常態分配假設，是許多統計方法的前提。
  • U型曲線（U-shaped Curve）：資料集中於兩個極端，中間較少。常見於政治立場的兩極化意見（強烈贊成 vs 強烈反對）。
  • J型曲線（J-shaped Curve）：正向或反向J型，資料嚴重偏向某一端。常見於財富分配、網站流量等冪次法則現象。

次數分配表

次數分配表（Frequency Distribution Table）是將資料整理成各類別或各組距出現次數的表格，是資料彙總的基本工具。通常包含：

  • 次數（Frequency）：各類別出現的個數
  • 相對次數（Relative Frequency）：各類別次數除以總次數（即百分比）
  • 累積次數（Cumulative Frequency）：由小到大累積的次數

📋 範例：某班 50 位同學對「對線上學習的整體滿意度」的調查結果（1=非常不滿意，5=非常滿意）

常見統計圖表類型

選擇正確的圖表是資料溝通的關鍵。以下介紹社會科學研究最常用的六種圖表，說明各自的適用情境、視覺特徵與限制。

① 圓餅圖（Pie Chart）

以扇形面積（角度）呈現各類別佔整體的比例，強調「部分對整體」的關係。
  • 適用：類別數少（2–4個）、各類別比例差異明顯、重點是「佔比」而非「比較大小」
  • 不適用：類別超過5個、需精確比較相近比例、需呈現時間趨勢
⚠️ 常見批評：人眼對角度與面積的判斷不如對長度精確。兩個扇形若比例相近（如22% vs 25%），視覺上難以分辨。多數情況下，長條圖的表達效果優於圓餅圖。

以「五數摘要」壓縮呈現連續變數的分配，特別擅長多組比較與偵測離群值，是社會科學最常用的探索性圖表之一。
圖 5-4　箱形圖各部位說明（IQR = 四分位距，涵蓋中間 50% 的資料）
  • 五數摘要：最小值（下鬚端）、Q1（25th percentile）、中位數、Q3（75th percentile）、最大值（上鬚端）
  • IQR（四分位距）：Q3 − Q1，涵蓋中間 50% 的資料，不受極端值影響
  • 離群值：超出 Q1 − 1.5×IQR 或 Q3 + 1.5×IQR 的觀測值，以獨立點標示
  • 最適合：多組資料的分配比較（如三種教學法的成績對比）；快速識別偏態與離群值

💡 箱形圖 vs 直方圖：直方圖看單組的分配形狀；箱形圖看多組之間的比較。兩者互補，大型研究通常兩種都呈現。

⑤ 折線圖（Line Chart）

⑥ 散布圖（Scatter Plot）

UNIT 06
集中趨勢與離散趨勢

眾數（Mode）

眾數是資料中出現次數最多的數值，適用於所有尺度，尤其是名目尺度。

眾數可能有一個（單峰）、兩個（雙峰）或多個；若每個數值只出現一次，則無眾數。

📐 計算範例

資料：3, 5, 7, 5, 9, 5, 3, 7, 5, 8
各值出現次數：3→2次　5→4次　7→2次　8→1次　9→1次
∴ 眾數 = 5（出現4次，最多）
💡 適用情境：名目尺度（如：最多人選擇的城市、最常見的職業類別）；或描述「最流行」、「最普遍」的狀況。
⚠️ 限制：不考慮所有數值，易受資料分組方式影響。

中位數（Median）

中位數是將資料由小到大排列後，位於正中間位置的數值。不受極端值影響，適合偏態分配。
  • 奇數個（n 為奇數）：中位數 = 第 (n+1)/2 個值
  • 偶數個（n 為偶數）：中位數 = 第 n/2 與 n/2+1 個值的平均

📐 計算範例（奇數）

原始資料：7, 3, 9, 5, 1
排序後：1, 3, 5, 7, 9　（n=5，取第3個）
∴ 中位數 = 5

📐 計算範例（偶數）

原始資料：4, 8, 2, 6, 10, 14
排序後：2, 4, 6, 8, 10, 14　（n=6，取第3、4個）
中位數 = (6 + 8) / 2 = 7
💡 適用情境：資料有極端值時（如薪資、房價、家庭所得）；或順序尺度資料。
⚠️ 限制：未利用所有數值的大小資訊，對小樣本較不穩定。

平均數（Mean）

算術平均數是所有數值的總和除以個數，是最常用的集中趨勢量數，但易受極端值影響。
母體平均數

μ = ΣXᵢ / N

N = 母體所有個體數
樣本平均數

X̄ = Σxᵢ / n

n = 樣本個數

📐 計算範例

資料：60, 70, 80, 90, 50
Σxᵢ = 60 + 70 + 80 + 90 + 50 = 350
n = 5
∴ X̄ = 350 / 5 = 70
💡 適用情境：計量尺度（區間/比例）、無極端值、接近對稱分配時最可靠。
⚠️ 極端值警告：資料：60, 70, 80, 90, 500 → 平均數 = 160，嚴重偏離大多數人的狀況！此時應改用中位數。

三種集中趨勢的比較與選用

量數 計算方式 受極端值影響 適用尺度 最佳使用情境
眾數 出現最多次的值 ✓ 不受影響 名目、順序、計量 類別資料、最流行的選項
中位數 排序後的中間值 ✓ 不受影響 順序、計量 有極端值、偏態分配（薪資、房價）
平均數 總和 ÷ 個數 ✗ 易受影響 計量（區間、比例） 對稱分配、無極端值、需進一步統計分析

偏態分配對集中趨勢的影響

當資料呈現偏態時，三種量數的大小關係會有規律性的變化：

圖 6-1　偏態分配對平均數、中位數、眾數的影響

離散趨勢：變異數與標準差

集中趨勢告訴我們資料的「中心」，但兩組資料可能平均數相同，卻分散程度截然不同。離散趨勢用來描述資料的「分散程度」。

圖 6-2　兩組平均數相同（70分），但標準差差異懸殊

變異數（Variance）

變異數是每個數值與平均數之差的平方的平均，衡量資料的整體分散程度。
平方的目的：消除正負號，讓偏差可以加總。
母體變異數

σ² = Σ(Xᵢ−μ)² / N

σ 讀作「sigma」
樣本變異數

s² = Σ(xᵢ−X̄)² / (n−1)

分母用 n−1（自由度修正，使估計更準確）

標準差（Standard Deviation）

標準差是變異數的正平方根，單位與原始資料相同，是最常用的離散量數。
母體標準差

σ = √σ²

= √[ Σ(Xᵢ−μ)² / N ]
樣本標準差

s = √s²

= √[ Σ(xᵢ−X̄)² / (n−1) ]

📐 完整計算範例（5位學生的統計成績）

學生 成績 xᵢ xᵢ − X̄ (xᵢ − X̄)²
A6060 − 70 = −10100
B6565 − 70 = −525
C7070 − 70 = 00
D7575 − 70 = +525
E8080 − 70 = +10100
合計 Σ = 350 Σ = 0 ✓ Σ = 250
樣本平均數：X̄ = 350 / 5 = 70
樣本變異數：s² = 250 / (5−1) = 250 / 4 = 62.5
樣本標準差：s = √62.5 ≈ 7.91
（解讀：這5位學生的成績平均偏離均值約 7.91 分）
🔑 為何分母用 n−1？這稱為「自由度（Degrees of Freedom）」的概念。當我們用樣本估計母體時，已知 X̄ 後，只需知道 n−1 個數值就能推算出最後一個值。用 n−1 修正能讓樣本變異數成為母體變異數的不偏估計量（Unbiased Estimator）。
💡 標準差的解讀：標準差越大，資料越分散；越小，資料越集中。
在常態分配下：μ ± 1σ 涵蓋約 68%；μ ± 2σ 涵蓋約 95%；μ ± 3σ 涵蓋約 99.7%（即68-95-99.7法則）。

中央極限定理（概念）

當樣本數 n 夠大時，樣本平均數的抽樣分配會趨近於常態分配，不論原始母體的分配形狀如何。

實際意義：這就是為什麼 n=10 時的樣本平均數分配較為不規則，而 n=200 時幾乎呈現完美的鐘型曲線。樣本數越大，統計推論越可靠。

樣本平均數的標準差稱為標準誤（Standard Error）：
SE = σ / √n　→ n 越大，SE 越小，估計越精確。

UNIT 07
假設檢定與統計分析方法

虛無假設 vs 對立假設

虛無假設（H₀, Null Hypothesis）
預設「沒有效果」或「沒有差異」的假設，是研究者試圖否定的假設。

例：「使用A教學法與B教學法的學生成績沒有差異」
對立假設（H₁, Alternative Hypothesis）
與虛無假設相反，研究者希望支持的假設。

例：「使用A教學法的學生成績顯著高於B教學法」
⚠️ 重要邏輯：假設檢定的結果只有「拒絕 H₀」或「無法拒絕 H₀」，而非「接受 H₁」。我們永遠無法「證明」虛無假設為真，只能找到足夠的統計證據來拒絕它。

自由度（Degrees of Freedom, df）

自由度是指在計算統計量時，可以自由變動的數值個數。當某些限制條件已知（如已知平均數），就會損失一個自由度。

📐 直觀說明：假設5個數字的平均數 = 10，總和 = 50。

已知：x₁=8, x₂=12, x₃=9, x₄=7, x₅=?
→ x₅ 被迫等於 50−8−12−9−7 = 14，無法自由選擇
∴ 5個數值中只有 4個可以自由變動 → df = n−1 = 4
統計情境自由度公式原因說明
樣本變異數 s² df = n − 1 已知 X̄，最後一個數值不自由，損失1個df
單樣本 t 檢定 df = n − 1 估計1個平均數，損失1個df
獨立樣本 t 檢定 df = n₁ + n₂ − 2 兩組各損失1個df（各估計1個平均數）
成對樣本 t 檢定 df = n − 1 對差值 d 計算，只有 n 對，損失1個df
單因子 ANOVA 組間 k−1；組內 N−k k 組、N 個觀察值，估計 k 個組平均數
卡方檢定 df = (r−1)(c−1) r 列 c 欄的交叉表，各行列已知合計後不自由
母體變異數 σ² df = N（除以 N） μ 已知（非估計），所有值皆自由，不需修正
🔑 為何樣本要除以 n−1 而非 n？
用樣本平均數 X̄ 估計 μ 時，X̄ 本身已「順著」樣本走，使偏差 (xᵢ−X̄) 系統性地偏小。若除以 n，變異數會被低估；除以 n−1 可修正這個偏誤，讓 s² 成為 σ² 的不偏估計量（Unbiased Estimator）。樣本越小，這個修正越重要。

查表：t 分配、F 分配、χ² 分配

在手算或報告結果時，需要對照統計分配表來判斷臨界值。查表的基本步驟：

  • Step 1 確定檢定類型：t 檢定 → t 表；ANOVA → F 表；卡方 → χ² 表
  • Step 2 確定自由度（df）：依據樣本數與組數計算
  • Step 3 確定顯著水準（α）：通常為 0.05 或 0.01
  • Step 4 確定檢定方向：雙尾（two-tailed）或單尾（one-tailed）
  • Step 5 比較：若計算統計量 > 臨界值，拒絕 H₀
圖 7-0　t 分配雙尾檢定的拒絕域（df=20, α=0.05，臨界值 ±2.086）
💡 查表 vs 軟體報 p 值：現代統計軟體（SPSS、R、Python）都會直接輸出 p 值，不需手動查表。但理解查表邏輯有助於理解「p 值是如何被判斷為顯著」的本質。

t 檢定的三種類型

① 單樣本 t 檢定（One-sample t-test）

比較樣本平均數與某個已知或假設的數值（μ₀）是否有顯著差異。
  • 公式：t = (X̄ − μ₀) / (s / √n)，df = n − 1
  • 例：某班30人平均成績 74 分，全國平均 70 分（σ未知）→ 這班是否顯著高於全國？
X̄ = 74，μ₀ = 70，s = 8，n = 30
t = (74 − 70) / (8 / √30) = 4 / 1.461 ≈ 2.74
df = 29，查表 t*(0.05, 雙尾) = 2.045
|t| = 2.74 > 2.045 → p < 0.05，拒絕 H₀，顯著高於全國平均

② 獨立樣本 t 檢定（Independent-samples t-test）

比較兩組互相獨立的樣本，其平均數是否有顯著差異。兩組成員完全不同，互不影響。
  • 公式：t = (X̄₁ − X̄₂) / SE_pooled，df = n₁ + n₂ − 2
  • 前提：Levene's Test 先檢定兩組變異數是否相等（等變異 or 不等變異版本）
  • 例：男生（n=40, X̄=72）vs 女生（n=40, X̄=78）的統計成績是否有差異？

📋 範例資料與結果：

組別nX̄s
男生4072.09.5
女生4078.08.8
df = 40 + 40 − 2 = 78
計算所得 t = −2.95，p = 0.004 < 0.01
結論：女生平均成績（78分）顯著高於男生（72分），** p < 0.01

③ 成對樣本 t 檢定（Paired-samples t-test）

比較同一組受試者在兩個條件下（前/後、左/右、甲/乙）的差異，控制了個體差異。比獨立樣本 t 檢定統計力更高。
  • 核心邏輯：計算每對的差值 dᵢ = x_後 − x_前，再對 dᵢ 做單樣本 t 檢定
  • 公式：t = d̄ / (s_d / √n)，df = n − 1（n 為配對數）
  • 例：同一班學生上課前後的統計焦慮分數（1−10分）是否有改變？
學生課前分數課後分數差值 d = 後−前

同一批人測量兩次（前後測、重複量測）→ 成對 t 檢定
兩組完全不同的人（男生 vs 女生、實驗組 vs 控制組）→ 獨立樣本 t 檢定

單因子變異數分析（One-Way ANOVA）

當要比較三組或三組以上的平均數是否有差異時，使用 ANOVA。若用多次 t 檢定替代，會大幅膨脹型一錯誤率（α累積問題）。
💡 為何不能用多次 t 檢定替代？
3組需要做3次 t 檢定，每次型一錯誤率 α=0.05，累積錯誤率 = 1−(0.95)³ ≈ 14.3%，遠超可接受範圍。ANOVA 同時檢定所有組，維持整體 α=0.05。
  • H₀：所有組的母體平均數相等（μ₁ = μ₂ = μ₃ = …）
  • H₁：至少有一組的平均數不同
  • 統計量：F = 組間變異（Between-group）/ 組內變異（Within-group）
  • df：組間 df = k−1；組內 df = N−k（k=組數，N=總樣本數）

ANOVA 前提假設：（1）各組獨立；（2）各組資料近似常態分配；（3）各組變異數相等（Levene's Test 檢驗）。

卡方檢定（Chi-Square Test）

適用情境：檢驗兩個類別變數之間是否有關聯（獨立性檢定），或某個類別變數的分配是否符合預期（適合度檢定）。
  • 資料類型：名目尺度或順序尺度（類別資料）
  • 典型應用：性別與投票傾向是否有關聯？教育程度與政治立場是否相關？
  • 輸出結果：交叉表（Cross-tabulation），含 χ² 統計量與 p 值
  • 注意事項：各格期望次數（Expected Count）應 ≥ 5，否則結果不可靠

迴歸分析概念（Regression）

迴歸分析用來探討自變數（X）對因變數（Y）的影響方向與大小，並建立預測模型。與相關分析不同，迴歸有明確的「因果方向假設」。

簡單線性迴歸（Simple Linear Regression）

Ŷ = b₀ + b₁X + ε
  • Ŷ（Y-hat）：預測值（因變數）
  • b₀：截距（X=0 時 Y 的預測值）
  • b₁：斜率（X 每增加1單位，Y 平均增加 b₁ 單位）
  • ε：殘差（誤差項，實際值與預測值的差距）
圖 7-4　簡單線性迴歸：迴歸線、截距 b₀、斜率 b₁ 與殘差 ε

多元線性迴歸（Multiple Linear Regression）

Ŷ = b₀ + b₁X₁ + b₂X₂ + … + bₖXₖ + ε

同時控制多個自變數，觀察每個 X 對 Y 的獨立影響（排除其他變數的干擾）。

📋 範例：以「收入（月薪）」為因變數：

標準化係數 β：不同尺度的自變數都轉換成標準差單位，可直接比較各變數的相對影響力大小（|β| 越大，影響越強）。
⚠️ 迴歸 ≠ 因果：迴歸係數描述的是統計上的關聯性，不代表因果關係。例如「教育年數正向預測收入」並不排除「家庭背景」等混淆變數的存在。

UNIT 08
機率基礎概念

為什麼要學機率？

統計推論的核心問題是：「我抽到的樣本結果，有多大可能是偶然發生的？」回答這個問題，需要用到機率的語言。機率是連結「樣本觀察」和「母體推論」的橋樑。

機率（Probability）：某一事件發生的可能性大小，以 0 到 1 之間的數字表示。

P = 0　→　該事件不可能發生
P = 1　→　該事件必然發生
P = 0.5 → 類似丟銅板，各半的機率
💡 生活中的機率例子：選舉民調說「支持率 60%，誤差 ±3%」，這個「誤差範圍」就是用機率計算出來的。統計顯著性（p 值）也是一種機率——它告訴你：「如果虛無假設是對的，抽到這種結果的機率有多低？」

機率的三種理解方式

類型概念例子研究應用
古典機率 所有結果等可能發生
P = 有利結果數 / 全部結果數		 丟骰子出現6的機率 = 1/6 理論基礎
頻率機率 大量重複實驗後，事件發生的相對次數 投擲硬幣10,000次，正面約5,000次 問卷調查的比例
主觀機率 根據個人知識與經驗的主觀判斷 「我認為今年台灣GDP成長機率是70%」 貝氏統計的基礎
💡 貝氏統計（Bayesian Statistics）簡介：與傳統（頻率派）統計不同，貝氏統計允許研究者將事前知識（prior）納入分析，透過觀察到的資料更新對母體的信念，得到後驗分配（posterior）。例如：初步研究認為新藥有效的機率是 60%，收到新的臨床試驗結果後，可更新這個估計。貝氏統計在機器學習、醫學研究中應用廣泛，初等統計通常以頻率派方法為主，貝氏為進階選修內容。

常態分配：統計學的核心

常態分配（Normal Distribution）是統計學中最重要的機率分配，又稱「鐘型曲線」。大多數社會現象（身高、考試成績、民調支持率）都接近常態分配。如果抽樣會導致偏差時，將會進行加權(weight)來讓樣本膨脹或縮減到符合常態分布。

圖 8-1　常態分配與 68–95–99.7 法則
  • μ ± 1σ：涵蓋約 68% 的資料
  • μ ± 2σ：涵蓋約 95% 的資料（信賴區間的基礎）
  • μ ± 3σ：涵蓋約 99.7% 的資料
標準化（Z-score）：將任何常態分配轉換為「標準常態分配」（μ=0, σ=1）的過程。

Z = (X − μ) / σ

Z 值的絕對意義：「這個觀測值距離平均數有幾個標準差？」例如：Z = 2，代表高於平均數 2 個標準差，屬於前約 2.3%。

UNIT 09
抽樣方法與抽樣分配

為什麼抽樣方法很重要？

一份研究的結論是否能推論到母體，很大程度取決於抽樣方法是否公正。若樣本不具代表性，再精密的統計分析也無法挽救。

抽樣誤差（Sampling Error）：樣本統計量（如 X̄）與母體參數（如 μ）之間的差距。只要不做普查，抽樣誤差就必然存在，但可以透過增大樣本數來降低。
樣本數、誤差與研究成本的三角關係：
增大樣本數可以減少抽樣誤差，使估計更精確——但代價是時間、金錢與人力成本的上升。研究者需要在「精確度」與「可行成本」之間做出平衡。

▸ 樣本數加倍，誤差只能縮小約 30%（因誤差與 √n 成反比）
▸ 要將誤差縮小為原來的 1/2，樣本數需增加到原來的 4 倍
▸ 學術研究通常透過「事前檢定力分析（Power Analysis）」決定最低所需樣本數，以在有限預算內達到足夠的統計檢定力。
非抽樣誤差（Non-sampling Error）：與抽樣過程無關的錯誤，例如問卷設計不良、訪員偏誤、拒訪、測量工具問題等。增大樣本數無法改善非抽樣誤差，只能透過研究設計來減少。

常見抽樣方法

簡單隨機抽樣（Simple Random Sampling） 機率抽樣

母體中每個個體被選中的機率完全相等。最理想的抽樣方式，但需要完整的母體名單（抽樣框）。

例：從全校5000名學生名單中，以亂數表隨機抽出200人

系統抽樣（Systematic Sampling） 機率抽樣

確定抽樣間隔 k = N/n，隨機選定起始點後，每隔 k 個抽一個。

例：1000份問卷中要抽100份，k=10；從第7份起，每隔10份抽一份（7、17、27…）

分層抽樣（Stratified Sampling） 機率抽樣

先將母體依特定特徵（如年齡、性別）分成若干層（strata），再在各層內進行隨機抽樣。

例：研究台灣閱讀習慣，依年齡層分20-39、40-59、60+，各層按比例抽樣

叢集抽樣（Cluster Sampling） 機率抽樣

將母體分成若干群（cluster），隨機抽取部分群，再對被選中的群進行全面調查或再次抽樣。

例：研究台灣中小學生，先隨機抽20所學校，再對各校全體學生施測

立意抽樣（Purposive Sampling） 非機率抽樣

依研究者的判斷與目的，有意識地選取具代表性或特殊意義的個體。

例：研究媒體識讀，特意選取新聞科系、教育學系學生作為受訪者

便利抽樣（Convenience Sampling） 非機率抽樣

選取最容易取得的個體，通常用於探索性研究或前測，不適合推論母體。

例：在學校門口詢問路過的學生填問卷；用自己的 Facebook 好友作為研究對象

💡 量化研究論文的關鍵：只有機率抽樣才能進行統計推論（把結果推廣到母體）。若使用便利抽樣，結論只能描述這批受訪者，論文中需明確說明此限制。

抽樣分配與中央極限定理

假設我們重複從母體中抽取樣本，每次計算樣本平均數 X̄，把所有這些 X̄ 的分配畫出來，就得到「抽樣分配（Sampling Distribution）」。

圖 9-1　中央極限定理：不論母體形狀，樣本數夠大時 X̄ 的分配趨近常態
中央極限定理（Central Limit Theorem, CLT）：
無論母體的分配形狀為何，當樣本數 n 夠大（通常 n ≥ 30），樣本平均數 X̄ 的抽樣分配會趨近於常態分配。

X̄ 的抽樣分配特性：
   • 平均而言，X̄ 的期望值 = μ（即在大量重複抽樣下，樣本平均數的均值趨近母體均值）
   • 標準差 = σ / √n　→ 稱為標準誤（Standard Error, SE）

重要意義：n 越大，SE 越小，X̄ 越接近 μ，估計越精確。這也說明了為什麼大樣本的研究更可信。
標準誤（SE）= σ / √n
n 增加為原來的 4 倍，SE 縮小為原來的 1/2
標準差（Standard Deviation, SD）
描述個別觀測值與平均數之間的平均距離。
是資料本身離散程度的量測。

▸ 對象：原始資料點
▸ 公式：s = √[Σ(xᵢ−X̄)² / (n−1)]
▸ 例：「成績的標準差 = 10 分」
標準誤（Standard Error, SE）
描述樣本平均數 X̄ 的抽樣變異程度。
是估計量精確度的量測。

▸ 對象：樣本平均數
▸ 公式：SE = s / √n
▸ 例：「平均成績的標準誤 = 1.4 分」
💡 觀念釐清標準差描述「資料有多散」；標準誤描述「我的估計（X̄）有多準」。樣本數越大，標準誤越小，X̄ 越接近真實 μ，但標準差不會因樣本數增加而縮小。

UNIT 10
統計估計與信賴區間

點估計 vs 區間估計

推論統計的第一步，是用樣本來估計母體的未知參數。估計有兩種形式：

點估計（Point Estimation）
用單一數值估計母體參數。

例：「根據問卷，學生平均每天閱讀 2.3 小時」

優點：直覺簡單
缺點：無法知道這個估計值有多可靠
區間估計（Interval Estimation）
提供一個範圍，並說明對這個範圍的信心程度。

例：「根據統計，在95% 信賴區間內，學生每日閱讀為 2.1 到 2.5 小時」

優點：呈現不確定性，更誠實
缺點：較複雜，需要解釋

信賴區間（Confidence Interval, CI）

信賴區間是統計報告中最常見的概念之一，也最常被誤解。

95% 信賴區間的正確解釋：
如果我們重複進行 100 次同樣的抽樣流程，每次計算一個信賴區間，其中大約 95 個區間會包含真實的母體參數 μ。
⚠️ 常見錯誤解釋：「母體參數有 95% 的機率落在這個區間內」—— 這是錯的！母體參數是固定的（只是我們不知道），不存在機率問題。是「這個流程構造的區間」有 95% 的機率包含參數，而非「這個特定區間」有 95% 的機率。
📝 學術論文寫法範例：

【方法段】
「本研究以獨立樣本 T 檢定比較兩組平均數，並以 Bootstrap 法（重複抽樣 1,000 次）構建 95% 信賴區間。」

【結果段】
「結果發現，有補習組的期末成績（M = 78.2, SD = 8.4）顯著高於未補習組（M = 72.1, SD = 10.3），t(48) = 2.31, p = .025, 95% CI [0.83, 11.37]，Cohen's d = 0.65。」

解讀：信賴區間 [0.83, 11.37] 表示在此研究設計下，有補習組與未補習組的成績差距之 95% CI 不包含 0，支持兩組確實有差異的結論。
95% 信賴區間公式（大樣本，σ 已知）：
X̄ ± 1.96 × (σ / √n)
1.96 是標準常態分配的 95% 臨界值（Z值）

若 σ 未知（通常如此），用樣本標準差 s 和 t 分配：
X̄ ± t(α/2, n-1) × (s / √n)
圖 10-1　95% 信賴區間的概念：100次抽樣中約95次的CI會包含真實μ
（圖中紅色虛線代表「真實母體參數 μ」——即我們想估計但永遠無法直接測量到的真實值。每條線段代表一次抽樣所構建的信賴區間；綠色線段包含了真實 μ，紅色線段則沒有，約佔 5%。）
R 語法示範 — 計算信賴區間
# 以第一份講義的成績資料為例
set.seed(42)
scores <- round(rnorm(50, mean = 75, sd = 10))

# 方法一：用 t.test 直接取得信賴區間
result <- t.test(scores)
cat("95% 信賴區間：", result$conf.int, "\n")
cat("樣本平均數：", round(mean(scores), 2), "\n")

# 方法二：手動計算（理解公式）
n    <- length(scores)
xbar <- mean(scores)
s    <- sd(scores)
se   <- s / sqrt(n)                          # 標準誤
t_crit <- qt(0.975, df = n - 1)              # t 臨界值（自由度 n-1）
ci_lower <- xbar - t_crit * se
ci_upper <- xbar + t_crit * se
cat("手動計算 95% CI: [", round(ci_lower,2), ",", round(ci_upper,2), "]\n")

UNIT 11
假設檢定深入

假設檢定的完整邏輯流程

假設檢定（Hypothesis Testing）的核心精神是：用樣本資料，判斷某種主張是否有足夠的統計證據支持。

1
建立假設（H₀ 和 H₁）

虛無假設 H₀ 是「無差異、無效果」的預設立場；對立假設 H₁ 是研究者想要支持的主張。
例：H₀: μ = 70（班級平均成績等於70分）；H₁: μ ≠ 70（不等於70分）

2
設定顯著水準 α

通常設為 α = 0.05（5%），代表「若 H₀ 為真，願意接受 5% 的犯第一型錯誤機率」。
嚴格研究可設 α = 0.01；探索性研究有時用 α = 0.10。

3
選擇適當的檢定統計量

根據研究問題與資料類型選擇：t 檢定（平均數）、卡方檢定（類別）、F 檢定（ANOVA）等。計算出檢定統計量的數值（t 值、χ² 值等）。

4
計算 p 值

p 值是「在 H₀ 成立的情況下，出現目前樣本結果（或更極端結果）的機率」。R 軟體會自動計算。

5
做出決策並解釋

若 p ＜ α → 拒絕 H₀，結果「統計上顯著」
若 p ≥ α → 無法拒絕 H₀，結果「未達顯著」
⚠️ 注意：「無法拒絕」≠「H₀ 是對的」，只是「證據不足」

📚 Type I / Type II Error 複習（第一講 Unit 04 已介紹）：本講補充統計檢定力（Statistical Power = 1 − β），即正確偵測到真實效果的機率。

提高統計檢定力的三種方法：① 增大樣本數（最有效）　② 提高顯著水準 α　③ 降低測量誤差

單尾 vs 雙尾檢定

雙尾檢定（Two-tailed）
H₁: μ ≠ μ₀（只問「有沒有差異」，不預設方向）

例：「新舊教材的學習效果有沒有差異？」

α 平均分配在左右兩端（各 α/2）
最常用，較保守
單尾檢定（One-tailed）
H₁: μ > μ₀ 或 μ < μ₀（預設方向）

例：「新教材學習效果優於舊教材？」

α 全部集中在單邊
較容易達到顯著，但需要理論依據
⚠️ 論文寫作建議：除非有充分的理論基礎支持特定方向，否則一律使用雙尾檢定，以避免被質疑「事後選擇有利的方向」（p-hacking）。

統計顯著 ≠ 實質重要

這是量化研究最常被誤解的地方之一，也是近年統計學界最熱議的議題。

效果量（Effect Size）：描述效果實際大小的指標，不受樣本數影響。

常用效果量：
  • Cohen's d：兩組平均數差異的標準化量度，用於 t 檢定
  • η²（eta-squared）：自變數解釋因變數總變異的比例，用於 ANOVA
  • R²：迴歸模型的解釋力（後述）
效果量 Cohen's d解釋例子
d ≈ 0.2小效果差異存在但不顯著，需要大樣本才能偵測
d ≈ 0.5中效果肉眼可以觀察到的差異
d ≈ 0.8大效果明顯的差異，實質意義較大
💡 實例：某廣告效果研究有 n=10,000，發現「看廣告組」購買意願比「未看組」高 0.1 分（5分量表），p=0.001（統計上非常顯著）。但 d=0.05，效果量極小，實質意義幾乎可忽略。

建議：論文中應同時報告 p 值與效果量。
R 語法示範 — 計算效果量 Cohen's d
# 安裝效果量計算套件
install.packages("effsize")
library(effsize)

# 沿用第一份講義的補習資料
tutoring_yes <- scores |> filter(tutoring == "有補習") |> pull(score)
tutoring_no  <- scores |> filter(tutoring == "未補習") |> pull(score)

# t 檢定 + 效果量
t.test(tutoring_yes, tutoring_no)
cohen.d(tutoring_yes, tutoring_no)

# 解讀：d 的大小對應「小/中/大」效果
d <- cohen.d(tutoring_yes, tutoring_no)$estimate
cat("Cohen's d =", round(abs(d), 3),
    ifelse(abs(d) >= 0.8, "→ 大效果",
    ifelse(abs(d) >= 0.5, "→ 中效果", "→ 小效果")), "\n")

  • H₁: 使用時間越多 → 睡眠分數越低（即「更差」）
  • 轉換為 H₁: μ₁（高使用） > μ₂（低使用） → t 值為正 → 右尾檢定

⚠️ 重要提醒：除非有充分理論依據，論文中建議使用雙尾檢定，以避免「事後選邊」的質疑。

簡單線性迴歸分析

迴歸分析的目的是用一個（或多個）自變數 X，預測因變數 Y，並量化 X 對 Y 的影響大小。

簡單線性迴歸方程式：
Ŷ = b₀ + b₁X
Ŷ = 預測值　b₀ = 截距（X=0 時的 Y）　b₁ = 斜率（X 每增加1單位，Y 平均變化量）
斜率 b₁ 的解讀：
控制其他條件不變，X 每增加 1 個單位，Y 平均增加（或減少）b₁ 個單位。

例：b₁ = 2.5（教育年數 → 月薪）
解讀：教育年數每多 1 年，月薪平均增加 2.5 萬元
截距 b₀ 的解讀：
當 X = 0 時，Y 的預測值。

實際意義上常常沒有意義（例如「教育年數=0」不存在），主要是模型的數學基準點，不要過度解釋。

r Pearson 相關係數 −1 到 +1 相關分析（兩個計量變數的線性關聯） 「X 與 Y 的關聯方向和強度如何？」
R² 決定係數（r 的平方） 0 到 1 迴歸分析（模型解釋力） 「X 能解釋 Y 多少百分比的變異？」
💡 兩者的關係：在簡單線性迴歸（只有一個自變數）中，R² = r²（相關係數的平方）。例如 r = 0.8 → R² = 0.64，代表 X 解釋了 Y 64% 的變異。在複迴歸中，R² 是多個 X 共同解釋 Y 的比例，不能還原成單一 r 值，且要改看 Adjusted R²（調整後 R²）。

R² 與迴歸結果解讀

R²（決定係數）：自變數 X 能解釋因變數 Y 總變異的比例，範圍 0 到 1。

R² = 0.65 → 解讀：X 可以解釋 Y 變異的 65%，其餘 35% 由模型未納入的因素決定。

R² 越接近 1，模型擬合越好。但 R² 高不等於模型正確——可能是過度擬合（overfitting）。

UNIT 13
複迴歸分析入門

為什麼需要複迴歸？

現實世界中，因變數通常受到多個因素同時影響。複迴歸允許我們在控制其他變數的情況下，評估每個自變數的獨立效果。

💡 例：研究「月薪」的影響因素，若只做簡單迴歸「教育年數 → 月薪」，可能忽略了「工作年資」也影響月薪，造成遺漏變數偏誤（Omitted Variable Bias）。

複迴歸同時放入「教育年數」和「工作年資」，就能得到各自獨立的效果。
複迴歸方程式：
Ŷ = b₀ + b₁X₁ + b₂X₂ + b₃X₃ + … + ε
b₁ 的解讀：控制其他所有自變數不變，X₁ 每增加 1 單位，Y 平均變化 b₁ 單位

係數解讀：「其他條件相同」的概念

複迴歸係數最關鍵的解讀原則：

虛擬變數（Dummy Variable）

類別變數（如性別、職業類別、是否有補習）無法直接放入迴歸，需要先轉換為虛擬變數（0/1 編碼）。

規則：k 個類別的變數，需要建立 k−1 個虛擬變數（其中一個作為參照組）。

例：「教育程度」有 3 個類別（高中以下、大學、研究所以上）
→ 以「高中以下」為參照組
→ 建立 D₁（大學=1, 其他=0）和 D₂（研究所以上=1, 其他=0）

解讀：D₁ 的係數 = 大學學歷者相較於高中以下學歷者，月薪的平均差異

迴歸分析的使用前提（假設）

  • 線性關係：X 與 Y 之間存在線性關係（散佈圖應呈直線型）
  • 殘差常態性：殘差（e = Y − Ŷ）應接近常態分配（分位圖（英語：quantile-quantile plot）又稱QQ圖（Q-Q plot，Q代表分位數））
  • 殘差同質性（等變異性）：殘差的變異不應隨 X 的大小而改變（殘差 vs 預測值圖應無漏斗形）
  • 殘差獨立性：殘差之間不應有相關（避免時間序列資料的自我相關）
  • 無嚴重多元共線性（複迴歸）：自變數之間不應高度相關（VIF 值不宜超過 10）

執行 plot(model) 可得到四張診斷圖；若有 car 套件，執行 vif(model) 可檢驗多元共線性。
