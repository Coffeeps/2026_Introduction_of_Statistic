UNIT 01
統計學導論

統計學是什麼？

統計學是關於對社會經濟現象數據資料的獲取、整理、分析、描述和推斷方法的總稱。包括獲取數據資料的方法，以及妥善利用這些數據的方法。

統計學在研究中扮演的核心角色：幫助研究者從有限的資料中，做出有根據的判斷與推論。日常生活中隨處可見統計的應用——從選舉民調、醫療臨床試驗，到行銷調查，皆仰賴統計方法。

敘述統計 vs 推論統計

敘述統計（Descriptive Statistics）

如果我們的興趣只限於手頭現有的數據，而不準備把結果用來推論群體，則稱為敘述統計。

例：消費者問卷的調查結果本身
推論統計（Inferential Statistics）

任何對數據（即樣本）的處理導致預測或推論群體的統計，稱為推論統計。

例：利用消費者問卷的調查結果，預測未來會購買的商品
💡 著名案例——尿布與啤酒：超市透過分析龐大的銷售資料，發現「購買嬰兒尿布的消費者，常常同時購買啤酒」，從而調整貨架陳列，提升銷售額。這是推論統計應用在商業決策的經典實例。

統計與科學方法

科學研究遵循以下循環步驟：

  • 觀察（Observation）：科學家蒐集、研究與其問題有關的事實，因而必須觀察研究對象發生的情形。
  • 假說（Hypothesis）：科學家解釋所觀察的事實，然後將其預感或直覺轉化為假說或理論，以表達洞察資料的結果。
  • 預測（Prediction）：科學家從假說及其相關理論中推演出某些結果。若假說成立，則此結果是從假說得出的新知識。
  • 驗證（Verification）：科學家蒐集新事實（即觀察）去檢驗由假說或理論所推測的結果是否成立。這個步驟使上述三個步驟必須重複進行。

UNIT 02
母體與樣本

核心概念定義

母體（Population）：它是一群具有某種共同特性的人或事物之個體所構成的群體。每一群體通常含有多種特性，可依某種特性歸納為一母體，亦可依另一種特性歸納為另一母體。
樣本（Sample）：從母體中依照一定方法抽取的部分個體，用來代表並推論母體特性。樣本的品質取決於抽樣方法的公正性與代表性。
圖 2-1　母體與樣本之關係圖

統計變數：母體參數 vs 樣本統計量

母體參數（Parameter）
用來代表母體特徵的量值。

符號：μ（mu）

例：112年度A醫院所有新生兒之平均體重
樣本統計量（Statistic）
用來代表樣本特徵的量值。

符號：X̄（X-bar）

例：從A醫院新生兒中隨機抽出200位之平均體重
🔑 核心邏輯：我們通常無法直接量測母體（成本太高或不可能），因此透過抽取樣本、計算統計量（X̄），來估計母體參數（μ）的真實值。這正是統計推論的精髓。

UNIT 03
統計變數與測量尺度

變數與變量

統計資料一般是由一個或多個變數的值所組成：

  • 變數（Variable）：一種具有不同值或結果的特徵之衡量，凡一切可計量的特徵都稱為變數。例：身高、性別、教育程度、收入。
  • 變量（Value）：變數的值或量稱為變量。例：身高的變量可以是160cm、175cm等具體數值。

自變數 vs 因變數

自變數（Independent Variable）
被認為是「因」（cause），或用來解釋其他變數的變數。

在 y = f(x) 中，x 為自變數。
因變數（Dependent Variable）
被認為是「果」（effect），或是被解釋的變數。

在 y = f(x) 中，y 為因變數。
收入（因變數） = 教育程度 + 職業類型 + 性別 + 居住地區 + ε

四種測量尺度（Scale）

尺度決定了我們可以對資料進行哪些數學運算，也決定了應使用哪些統計方法。

尺度 類型 可用運算 特性 範例
名目尺度
Nominal		 非計量 = ≠ 僅能區分類別，無大小順序 性別、職業、語言、居住地區
順序尺度
Ordinal		 非計量 = ≠ > < 有大小順序，但間距不相等 教育程度（小學＜國中＜高中＜大學）
區間尺度
Interval		 計量 = ≠ > < + − 間距相等，但無絕對零點 顧客滿意度、溫度（℃）
（行銷學中假設滿意度各尺度間皆為等距）
比例尺度
Ratio		 計量 = ≠ > < + − × ÷ 有絕對零點，可進行所有四則運算 價格、年齡、所得、體重

  • 為什麼比例尺度才可以說「A是B的兩倍」，而區間尺度不行？

UNIT 04
誤差、顯著性與信效度

常態分配與68-95-99.7法則

許多自然與社會現象的資料分布呈現鐘型曲線（Bell-shaped Curve），也稱常態分配（Normal Distribution）。此分配具有以下特性：

  • 以平均數為中心，左右對稱
  • 平均數 = 中位數 = 眾數
  • 曲線下方面積代表機率（總面積 = 1）
圖 4-1　常態分配的68-95-99.7法則

p 值與統計顯著性

大家習慣採用 0.05 當作一個臨界，當研究的 p 值小於這個臨界值的時候就宣稱研究結果達到統計顯著，也就是說大家普遍同意接受 5% 犯錯的可能性。
100% − 5% 犯錯 = 95% 信心水準

這個願意接受的犯錯可能性稱為「顯著水準（Significant Level）」——超過這個水準就宣稱研究達到統計顯著。

符號意義信心水準
*p < 0.05達到基本顯著水準95%
**p < 0.01高度顯著99%
***p < 0.001極高度顯著99.9%
🧠 批判性思考
  • p 值越小，表示該研究的結果越好嗎？
  • 樣本數越大，研究的 p 值一定越小嗎？

型一錯誤 vs 型二錯誤

假設飛機的鉚釘「足夠強壯」為虛無假設（H₀），以下說明四種結果：

真實情況：鉚釘足夠強
真實情況：鉚釘不夠強
測試結果
✓ True Positive（真陽性）
測試結果：足夠強。這是我們希望看到的結果，無法拒絕虛無假設（正確）。
✗ False Positive（偽陽性）
型一錯誤 Type I Error (α)
測試結果：足夠強，但實際上不夠強。錯誤地無法拒絕虛無假設，認為鉚釘足夠強，但實際上不夠強壯。
✗ False Negative（偽陰性）
型二錯誤 Type II Error (β)
測試結果：不夠強，但實際上足夠強。錯誤地拒絕了虛無假設，認為鉚釘不夠強，但實際上是夠強壯的。
✓ True Negative（真陰性）
測試結果：不夠強。這是我們希望看到的結果，正確地拒絕了虛無假設。

信度與效度

信度（Reliability）
測量的穩定性與一致性。若重複測量同一對象，每次結果都相近，則信度高。

類比：每次射擊都落在同一區域（不論是否命中靶心）。
效度（Validity）
測量工具是否真正測量到它想測量的概念（準確性）。

類比：射擊是否命中靶心，是否測量到真正想測量的東西。

  • （A）信度低、效度低
  • （B）信度高、效度低
  • （C）信度低、效度高
  • （D）信度高、效度高
Q4.請說明：為什麼「樣本數越大，p 值不一定越小」？
參考答案：p 值反映的是「在虛無假設為真的情況下，觀察到這樣極端結果的機率」。樣本數增大確實會提高統計檢定力，使真實存在的效果更容易被偵測到（p 值降低）；但若兩組之間本來就沒有差異，無論樣本數多大，p 值都不會顯著降低。換言之，p 值取決於效果的大小與樣本數的共同作用，而非單純由樣本數決定。
UNIT 05
彙總資料：表格與圖表

民意曲線的常見形態

在社會科學研究中，資料分布常呈現以下幾種典型形態：

  • 鐘型曲線（Bell-shaped Curve）：正態分配，資料集中於中間，兩端逐漸減少。符合常態分配假設，是許多統計方法的前提。
  • U型曲線（U-shaped Curve）：資料集中於兩個極端，中間較少。常見於政治立場的兩極化意見（強烈贊成 vs 強烈反對）。
  • J型曲線（J-shaped Curve）：正向或反向J型，資料嚴重偏向某一端。常見於財富分配、網站流量等冪次法則現象。
圖 5-1　三種常見的民意曲線形態（以五點量表意見為例）

次數分配表

次數分配表（Frequency Distribution Table）是將資料整理成各類別或各組距出現次數的表格，是資料彙總的基本工具。通常包含：

  • 次數（Frequency）：各類別出現的個數
  • 相對次數（Relative Frequency）：各類別次數除以總次數（即百分比）
  • 累積次數（Cumulative Frequency）：由小到大累積的次數

📋 範例：某班 50 位同學對「對線上學習的整體滿意度」的調查結果（1=非常不滿意，5=非常滿意）

滿意度評分 類別說明 次數 (f) 相對次數 (%) 累積次數 累積百分比 (%)
1 非常不滿意 3 6.0% 3 6.0%
2 不滿意 7 14.0% 10 20.0%
3 普通 15 30.0% 25 50.0%
4 滿意 18 36.0% 43 86.0%
5 非常滿意 7 14.0% 50 100.0%
合計 50 100.0% — —

常見統計圖表類型

選擇正確的圖表是資料溝通的關鍵。以下介紹社會科學研究最常用的六種圖表，說明各自的適用情境、視覺特徵與限制。

① 圓餅圖（Pie Chart）

以扇形面積（角度）呈現各類別佔整體的比例，強調「部分對整體」的關係。
  • 適用：類別數少（2–4個）、各類別比例差異明顯、重點是「佔比」而非「比較大小」
  • 不適用：類別超過5個、需精確比較相近比例、需呈現時間趨勢
⚠️ 常見批評：人眼對角度與面積的判斷不如對長度精確。兩個扇形若比例相近（如22% vs 25%），視覺上難以分辨。多數情況下，長條圖的表達效果優於圓餅圖。
圓餅圖示意（3種教學法）

② 長條圖（Bar Chart）與 ③ 直方圖（Histogram）

這兩種圖長相相似，但用途完全不同，是最常被混淆的圖表組合。

比較項目 直方圖（Histogram） 長條圖（Bar Chart）
呈現目的資料分布形狀各類別大小比較
橫軸變數連續數值（計量）類別（名目/順序）
條與條之間無間隔（數值連續）有間隔（類別獨立）
順序可否對調❌ 不可（有數值順序）✓ 可以（類別無固定順序）
典型用途成績分布、年齡分布各城市人口、各職業比例

延伸：堆疊長條圖（Stacked Bar Chart）適合展示李克特量表各選項的比例分布，可搭配正反向配色（紅→灰→綠）讓態度趨勢一目瞭然。

④ 箱形圖（Box Plot）

以「五數摘要」壓縮呈現連續變數的分配，特別擅長多組比較與偵測離群值，是社會科學最常用的探索性圖表之一。
圖 5-4　箱形圖各部位說明（IQR = 四分位距，涵蓋中間 50% 的資料）
  • 五數摘要：最小值（下鬚端）、Q1（25th percentile）、中位數、Q3（75th percentile）、最大值（上鬚端）
  • IQR（四分位距）：Q3 − Q1，涵蓋中間 50% 的資料，不受極端值影響
  • 離群值：超出 Q1 − 1.5×IQR 或 Q3 + 1.5×IQR 的觀測值，以獨立點標示
  • 最適合：多組資料的分配比較（如三種教學法的成績對比）；快速識別偏態與離群值
  • 進階搭配：箱形圖 + 原始資料點plot可同時看到個別觀測值的散布
💡 箱形圖 vs 直方圖：直方圖看單組的分配形狀；箱形圖看多組之間的比較。兩者互補，大型研究通常兩種都呈現。

⑤ 折線圖（Line Chart）

用連線表示連續時間點或有序類別上數值的變化趨勢，強調「走向」而非「大小比較」。
  • 適用：時間序列資料（歷年GDP、月銷售量）；成對樣本前後測的個別變化（配對折線圖）
  • 注意：若橫軸是名目尺度（無順序），不應使用折線圖——點之間的連線沒有意義

⑥ 散布圖（Scatter Plot）

以平面座標展示兩個連續變數之間的關係，每個點代表一筆觀測值。是相關分析與迴歸分析的視覺化基礎。
  • 適用：探索兩變數的線性/非線性關係、相關強度、離群值分布
  • 搭配：加入迴歸線（R：geom_smooth(method="lm")）直觀展示預測趨勢
  • 常見問題：點的重疊（overplotting）→ 可用半透明（alpha）或 散點圖 處理

UNIT 06
集中趨勢與離散趨勢

眾數（Mode）

眾數是資料中出現次數最多的數值，適用於所有尺度，尤其是名目尺度。

眾數可能有一個（單峰）、兩個（雙峰）或多個；若每個數值只出現一次，則無眾數。

📐 計算範例

資料：3, 5, 7, 5, 9, 5, 3, 7, 5, 8
各值出現次數：3→2次　5→4次　7→2次　8→1次　9→1次
∴ 眾數 = 5（出現4次，最多）
💡 適用情境：名目尺度（如：最多人選擇的城市、最常見的職業類別）；或描述「最流行」、「最普遍」的狀況。
⚠️ 限制：不考慮所有數值，易受資料分組方式影響。

中位數（Median）

中位數是將資料由小到大排列後，位於正中間位置的數值。不受極端值影響，適合偏態分配。
  • 奇數個（n 為奇數）：中位數 = 第 (n+1)/2 個值
  • 偶數個（n 為偶數）：中位數 = 第 n/2 與 n/2+1 個值的平均

📐 計算範例（奇數）

原始資料：7, 3, 9, 5, 1
排序後：1, 3, 5, 7, 9　（n=5，取第3個）
∴ 中位數 = 5

📐 計算範例（偶數）

原始資料：4, 8, 2, 6, 10, 14
排序後：2, 4, 6, 8, 10, 14　（n=6，取第3、4個）
中位數 = (6 + 8) / 2 = 7
💡 適用情境：資料有極端值時（如薪資、房價、家庭所得）；或順序尺度資料。
⚠️ 限制：未利用所有數值的大小資訊，對小樣本較不穩定。

平均數（Mean）

算術平均數是所有數值的總和除以個數，是最常用的集中趨勢量數，但易受極端值影響。
母體平均數

μ = ΣXᵢ / N

N = 母體所有個體數
樣本平均數

X̄ = Σxᵢ / n

n = 樣本個數

📐 計算範例

資料：60, 70, 80, 90, 50
Σxᵢ = 60 + 70 + 80 + 90 + 50 = 350
n = 5
∴ X̄ = 350 / 5 = 70
💡 適用情境：計量尺度（區間/比例）、無極端值、接近對稱分配時最可靠。
⚠️ 極端值警告：資料：60, 70, 80, 90, 500 → 平均數 = 160，嚴重偏離大多數人的狀況！此時應改用中位數。

三種集中趨勢的比較與選用

量數 計算方式 受極端值影響 適用尺度 最佳使用情境
眾數 出現最多次的值 ✓ 不受影響 名目、順序、計量 類別資料、最流行的選項
中位數 排序後的中間值 ✓ 不受影響 順序、計量 有極端值、偏態分配（薪資、房價）
平均數 總和 ÷ 個數 ✗ 易受影響 計量（區間、比例） 對稱分配、無極端值、需進一步統計分析

偏態分配對集中趨勢的影響

當資料呈現偏態時，三種量數的大小關係會有規律性的變化：

圖 6-1　偏態分配對平均數、中位數、眾數的影響
💡 實際應用：薪資或房價等資料通常呈右偏分配（少數極高值拉高平均數）。此時用中位數報告比平均數更能反映「一般人」的狀況。

離散趨勢：變異數與標準差

集中趨勢告訴我們資料的「中心」，但兩組資料可能平均數相同，卻分散程度截然不同。離散趨勢用來描述資料的「分散程度」。

圖 6-2　兩組平均數相同（70分），但標準差差異懸殊

變異數（Variance）

變異數是每個數值與平均數之差的平方的平均，衡量資料的整體分散程度。
平方的目的：消除正負號，讓偏差可以加總。
母體變異數

σ² = Σ(Xᵢ−μ)² / N

σ 讀作「sigma」
樣本變異數

s² = Σ(xᵢ−X̄)² / (n−1)

分母用 n−1（自由度修正，使估計更準確）

標準差（Standard Deviation）

標準差是變異數的正平方根，單位與原始資料相同，是最常用的離散量數。
母體標準差

σ = √σ²

= √[ Σ(Xᵢ−μ)² / N ]
樣本標準差

s = √s²

= √[ Σ(xᵢ−X̄)² / (n−1) ]

📐 完整計算範例（5位學生的統計成績）

學生 成績 xᵢ xᵢ − X̄ (xᵢ − X̄)²
A6060 − 70 = −10100
B6565 − 70 = −525
C7070 − 70 = 00
D7575 − 70 = +525
E8080 − 70 = +10100
合計 Σ = 350 Σ = 0 ✓ Σ = 250
樣本平均數：X̄ = 350 / 5 = 70
樣本變異數：s² = 250 / (5−1) = 250 / 4 = 62.5
樣本標準差：s = √62.5 ≈ 7.91
（解讀：這5位學生的成績平均偏離均值約 7.91 分）
🔑 為何分母用 n−1？這稱為「自由度（Degrees of Freedom）」的概念。當我們用樣本估計母體時，已知 X̄ 後，只需知道 n−1 個數值就能推算出最後一個值。用 n−1 修正能讓樣本變異數成為母體變異數的不偏估計量（Unbiased Estimator）。
💡 標準差的解讀：標準差越大，資料越分散；越小，資料越集中。
在常態分配下：μ ± 1σ 涵蓋約 68%；μ ± 2σ 涵蓋約 95%；μ ± 3σ 涵蓋約 99.7%（即68-95-99.7法則）。

中央極限定理（概念）

當樣本數 n 夠大時，樣本平均數的抽樣分配會趨近於常態分配，不論原始母體的分配形狀如何。

實際意義：這就是為什麼 n=10 時的樣本平均數分配較為不規則，而 n=200 時幾乎呈現完美的鐘型曲線。樣本數越大，統計推論越可靠。

樣本平均數的標準差稱為標準誤（Standard Error）：
SE = σ / √n　→ n 越大，SE 越小，估計越精確。

  • A班標準差=5分，B班標準差=20分，兩班平均數都是75分。你會選哪班？為什麼？

UNIT 07
假設檢定與統計分析方法

虛無假設 vs 對立假設

虛無假設（H₀, Null Hypothesis）
預設「沒有效果」或「沒有差異」的假設，是研究者試圖否定的假設。

例：「使用A教學法與B教學法的學生成績沒有差異」
對立假設（H₁, Alternative Hypothesis）
與虛無假設相反，研究者希望支持的假設。

例：「使用A教學法的學生成績顯著高於B教學法」
⚠️ 重要邏輯：假設檢定的結果只有「拒絕 H₀」或「無法拒絕 H₀」，而非「接受 H₁」。我們永遠無法「證明」虛無假設為真，只能找到足夠的統計證據來拒絕它。

自由度（Degrees of Freedom, df）

自由度是指在計算統計量時，可以自由變動的數值個數。當某些限制條件已知（如已知平均數），就會損失一個自由度。

📐 直觀說明：假設5個數字的平均數 = 10，總和 = 50。

已知：x₁=8, x₂=12, x₃=9, x₄=7, x₅=?
→ x₅ 被迫等於 50−8−12−9−7 = 14，無法自由選擇
∴ 5個數值中只有 4個可以自由變動 → df = n−1 = 4
統計情境自由度公式原因說明
樣本變異數 s² df = n − 1 已知 X̄，最後一個數值不自由，損失1個df
單樣本 t 檢定 df = n − 1 估計1個平均數，損失1個df
獨立樣本 t 檢定 df = n₁ + n₂ − 2 兩組各損失1個df（各估計1個平均數）
成對樣本 t 檢定 df = n − 1 對差值 d 計算，只有 n 對，損失1個df
單因子 ANOVA 組間 k−1；組內 N−k k 組、N 個觀察值，估計 k 個組平均數
卡方檢定 df = (r−1)(c−1) r 列 c 欄的交叉表，各行列已知合計後不自由
母體變異數 σ² df = N（除以 N） μ 已知（非估計），所有值皆自由，不需修正
🔑 為何樣本要除以 n−1 而非 n？
用樣本平均數 X̄ 估計 μ 時，X̄ 本身已「順著」樣本走，使偏差 (xᵢ−X̄) 系統性地偏小。若除以 n，變異數會被低估；除以 n−1 可修正這個偏誤，讓 s² 成為 σ² 的不偏估計量（Unbiased Estimator）。樣本越小，這個修正越重要。

查表：t 分配、F 分配、χ² 分配

在手算或報告結果時，需要對照統計分配表來判斷臨界值。查表的基本步驟：

  • Step 1 確定檢定類型：t 檢定 → t 表；ANOVA → F 表；卡方 → χ² 表
  • Step 2 確定自由度（df）：依據樣本數與組數計算
  • Step 3 確定顯著水準（α）：通常為 0.05 或 0.01
  • Step 4 確定檢定方向：雙尾（two-tailed）或單尾（one-tailed）
  • Step 5 比較：若計算統計量 > 臨界值，拒絕 H₀
圖 7-0　t 分配雙尾檢定的拒絕域（df=20, α=0.05，臨界值 ±2.086）
💡 查表 vs 軟體報 p 值：現代統計軟體（SPSS、R、Python）都會直接輸出 p 值，不需手動查表。但理解查表邏輯有助於理解「p 值是如何被判斷為顯著」的本質。

t 檢定的三種類型

① 單樣本 t 檢定（One-sample t-test）

比較樣本平均數與某個已知或假設的數值（μ₀）是否有顯著差異。
  • 公式：t = (X̄ − μ₀) / (s / √n)，df = n − 1
  • 例：某班30人平均成績 74 分，全國平均 70 分（σ未知）→ 這班是否顯著高於全國？
X̄ = 74，μ₀ = 70，s = 8，n = 30
t = (74 − 70) / (8 / √30) = 4 / 1.461 ≈ 2.74
df = 29，查表 t*(0.05, 雙尾) = 2.045
|t| = 2.74 > 2.045 → p < 0.05，拒絕 H₀，顯著高於全國平均

② 獨立樣本 t 檢定（Independent-samples t-test）

比較兩組互相獨立的樣本，其平均數是否有顯著差異。兩組成員完全不同，互不影響。
  • 公式：t = (X̄₁ − X̄₂) / SE_pooled，df = n₁ + n₂ − 2
  • 前提：Levene's Test 先檢定兩組變異數是否相等（等變異 or 不等變異版本）
  • 例：男生（n=40, X̄=72）vs 女生（n=40, X̄=78）的統計成績是否有差異？

📋 範例資料與結果：

組別nX̄s
男生4072.09.5
女生4078.08.8
df = 40 + 40 − 2 = 78
計算所得 t = −2.95，p = 0.004 < 0.01
結論：女生平均成績（78分）顯著高於男生（72分），** p < 0.01

③ 成對樣本 t 檢定（Paired-samples t-test）

比較同一組受試者在兩個條件下（前/後、左/右、甲/乙）的差異，控制了個體差異。比獨立樣本 t 檢定統計力更高。
  • 核心邏輯：計算每對的差值 dᵢ = x_後 − x_前，再對 dᵢ 做單樣本 t 檢定
  • 公式：t = d̄ / (s_d / √n)，df = n − 1（n 為配對數）
  • 例：同一班學生上課前後的統計焦慮分數（1−10分）是否有改變？
學生課前分數課後分數差值 d = 後−前
A86−2
B75−2
C96−3
D64−2
E87−1
平均差值 d̄ = −2.0
d̄ = −2.0，s_d = 0.707，n = 5，df = 4
t = −2.0 / (0.707 / √5) = −2.0 / 0.316 ≈ −6.33
查表 t*(0.05, 雙尾, df=4) = 2.776
|t| = 6.33 > 2.776 → p < 0.05，課後統計焦慮顯著降低
🔍 獨立 vs 成對的選擇原則：
同一批人測量兩次（前後測、重複量測）→ 成對 t 檢定
兩組完全不同的人（男生 vs 女生、實驗組 vs 控制組）→ 獨立樣本 t 檢定

單因子變異數分析（One-Way ANOVA）

當要比較三組或三組以上的平均數是否有差異時，使用 ANOVA。若用多次 t 檢定替代，會大幅膨脹型一錯誤率（α累積問題）。
💡 為何不能用多次 t 檢定替代？
3組需要做3次 t 檢定，每次型一錯誤率 α=0.05，累積錯誤率 = 1−(0.95)³ ≈ 14.3%，遠超可接受範圍。ANOVA 同時檢定所有組，維持整體 α=0.05。
  • H₀：所有組的母體平均數相等（μ₁ = μ₂ = μ₃ = …）
  • H₁：至少有一組的平均數不同
  • 統計量：F = 組間變異（Between-group）/ 組內變異（Within-group）
  • df：組間 df = k−1；組內 df = N−k（k=組數，N=總樣本數）

翻轉教室2076.87.9
線上自學2073.19.2
F(2, 57) = 4.21，p = 0.020 < 0.05 → 三種教學法的成績有顯著差異（*）
⚠️ ANOVA 只告訴我們「至少有一組不同」，需進行事後比較（Post-hoc Test）（如 Tukey HSD）才能知道是哪兩組之間有差異。
ANOVA 前提假設：（1）各組獨立；（2）各組資料近似常態分配；（3）各組變異數相等（Levene's Test 檢驗）。

卡方檢定（Chi-Square Test）

適用情境：檢驗兩個類別變數之間是否有關聯（獨立性檢定），或某個類別變數的分配是否符合預期（適合度檢定）。
  • 資料類型：名目尺度或順序尺度（類別資料）
  • 典型應用：性別與投票傾向是否有關聯？教育程度與政治立場是否相關？
  • 輸出結果：交叉表（Cross-tabulation），含 χ² 統計量與 p 值
  • 注意事項：各格期望次數（Expected Count）應 ≥ 5，否則結果不可靠

📋 範例資料集：調查 200 位受訪者「性別」與「是否支持同性婚姻」：

性別 是否支持同性婚姻 列合計
支持 不支持
男性 55 (期望 62) 45 (期望 38) 100
女性 69 (期望 62) 31 (期望 38) 100
欄合計 124 76 200
圖 7-1　卡方檢定：實際次數 vs 期望次數視覺化（差距越大，χ² 值越大）
計算結果：χ²(1) = 3.96，p = 0.047 < 0.05，達統計顯著（*）。
結論：性別與是否支持同性婚姻之間有顯著關聯——女性支持比例（69%）顯著高於男性（55%）。

迴歸分析概念（Regression）

迴歸分析用來探討自變數（X）對因變數（Y）的影響方向與大小，並建立預測模型。與相關分析不同，迴歸有明確的「因果方向假設」。

簡單線性迴歸（Simple Linear Regression）

Ŷ = b₀ + b₁X + ε
  • Ŷ（Y-hat）：預測值（因變數）
  • b₀：截距（X=0 時 Y 的預測值）
  • b₁：斜率（X 每增加1單位，Y 平均增加 b₁ 單位）
  • ε：殘差（誤差項，實際值與預測值的差距）
圖 7-4　簡單線性迴歸：迴歸線、截距 b₀、斜率 b₁ 與殘差 ε

多元線性迴歸（Multiple Linear Regression）

Ŷ = b₀ + b₁X₁ + b₂X₂ + … + bₖXₖ + ε

同時控制多個自變數，觀察每個 X 對 Y 的獨立影響（排除其他變數的干擾）。

📋 範例：以「收入（月薪）」為因變數：

變數b（迴歸係數）β（標準化係數）p 值解讀
截距（b₀）12,000——其他條件為0時的基準收入
教育年數（X₁）+2,5000.420.001 **多讀1年，月薪增加2,500元
工作年資（X₂）+1,2000.310.008 **多1年年資，月薪增加1,200元
性別（女=1，X₃）−3,800−0.180.032 *控制教育與年資後，女性比男性少3,800元
R²（決定係數）：迴歸模型能解釋因變數變異的比例。R²=0.58 表示模型解釋了58%的收入差異，剩餘42%由其他未納入因素決定。

標準化係數 β：不同尺度的自變數都轉換成標準差單位，可直接比較各變數的相對影響力大小（|β| 越大，影響越強）。
⚠️ 迴歸 ≠ 因果：迴歸係數描述的是統計上的關聯性，不代表因果關係。例如「教育年數正向預測收入」並不排除「家庭背景」等混淆變數的存在。

Q3.某研究的虛無假設 H₀：「補習班對成績沒有影響」，結果 p = 0.02（α = 0.05）。請寫出正確的研究結論。
參考答案：由於 p = 0.02 ＜ α = 0.05，達到統計顯著水準，因此拒絕虛無假設（H₀）。結論為：有統計上的顯著證據顯示補習班對學生成績有影響（對立假設 H₁ 獲得支持）。

⚠️ 注意：統計顯著不代表「影響很大」，仍須考量效果量（Effect Size）來判斷實際意義。
Q4.【綜合應用】一份研究想探討：教育程度（高中以下、大學、研究所以上）對收入（月薪，元）的影響。請回答：
（1）自變數與因變數各是什麼尺度？
（2）應該使用什麼統計方法？
（3）如果要將「教育程度」放入迴歸模型，需要做什麼處理？
參考答案：
（1）自變數：教育程度 → 順序尺度（名目或順序皆可接受）；因變數：月薪 → 比例尺度
（2）可使用單因子變異數分析（One-way ANOVA）比較三組平均月薪是否有差異；若要控制其他變數，可使用多元迴歸分析
（3）「教育程度」為類別變數，放入迴歸前需建立虛擬變數（Dummy Variables）：以「高中以下」為參照組，建立2個虛擬變數（大學=1/0；研究所以上=1/0）
